# Portfolio Personal - Federico Fernández

## Descripción
Este proyecto es una landing page personal desarrollada como trabajo práctico integrador. Presenta información sobre mí, mis habilidades, proyectos y un formulario de contacto.

## Tecnologías utilizadas
- HTML
- CSS
- JavaScript
- Git y GitHub

## Funcionalidades
- Modo oscuro / claro
- Personalización del nombre
- Mostrar y ocultar proyectos
- Validación de formulario

## Cómo ejecutar el proyecto
1. Clonar el repositorio:
   git clone https://github.com/fedegon2211995/portfolio-federico-fernandez.git

2. Abrir el archivo index.html en el navegador
